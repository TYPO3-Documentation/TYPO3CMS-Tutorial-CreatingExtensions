Explanation
===========
:code:`t3crext` is the usual alias for :code:`https://docs.typo3.org/typo3cms/CreatingExtensionsTutorial`
when creating `cross-references
<http://www.sphinx-doc.org/en/stable/markup/inline.html#cross-referencing-arbitrary-locations>`__
(links) to this manual. *Example:* ``:ref:`Creating Extensions Tutorial <t3crext:start>```.
Have the following lines in your :file:`Documentation/Settings.cfg` file to set the alias::

   [intersphinx_mapping]

   t3crext = https://docs.typo3.org/typo3cms/CreatingExtensionsTutorial
